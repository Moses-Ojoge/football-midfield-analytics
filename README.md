# Football Midfield Analytics

This project analyzes midfielders across six top football leagues using **Python** and **NumPy**.  

It generates realistic player stats, normalizes the data, ranks leagues and players, and produces a professional report-style console output.

---

## Features

- Random realistic stats for 6 leagues, 10 midfielders per league, 8 metrics per player.
- Normalizes data to allow fair comparison across metrics.
- Identifies top 5 midfielders in:
  - Overall performance
  - Creative
  - Attacking
  - Defensive
- Identifies:
  - Most balanced player
  - Most balanced league
  - Most specialized league
- Produces league-level performance rankings and role-based insights.
- Report-style console output ready for scouting or portfolio showcase.

## Usage

1. Clone the repository:

```bash
git clone https://github.com/<your-username>/football-midfield-analytics.git
cd football-midfield-analytics
```

2. Run the analysis:

```bash
python midfield_analysis.py
```

You will see a full report of league and player insights in the console.
